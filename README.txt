StudentNest - starter website
Files:
- index.php, about.php, courses.php, services.php, blog.php, faq.php
- register.php, login.php, dashboard.php, contact.php, logout.php
- includes/header.php, includes/footer.php
- assets/css/styles.css, assets/js/scripts.js
- data/users.json, data/messages.json

Instructions:
1. Copy the studentnest folder to your PHP-enabled server (e.g., XAMPP's htdocs or a real web host).
2. Ensure the data/ folder is writable by PHP (permissions).
3. Open http://localhost/studentnest/index.php to view.
